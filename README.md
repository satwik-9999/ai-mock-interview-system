# AI Mock Interview System

## Overview

AI Mock Interview System is a backend application built using Spring Boot and REST APIs. The project simulates technical interview preparation by generating role-based interview questions dynamically through API endpoints.

## Technologies Used

* Java
* Spring Boot
* REST API
* Maven
* IntelliJ IDEA
* Postman

## Features

* Generate interview questions based on role
* REST API endpoint handling
* Dynamic request parameter support
* Backend API architecture using Spring Boot

## API Endpoint

### Generate Interview Question

GET `/interview?role=Java Developer`

### Example Response

Generated Interview Question for Java Developer: Explain the difference between HashMap and ConcurrentHashMap.

## Project Architecture

Client Request → Spring Boot Controller → API Response

## Testing

API endpoints were tested using browser and Postman.

## Learning Outcomes

* Spring Boot backend development
* REST API creation
* Request parameter handling
* Controller architecture
* API testing using Postman
